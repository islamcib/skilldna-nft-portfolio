NFT Портфолио Навыков
Ваши навыки в блокчейне Ethereum

✅ СИСТЕМА АКТИВНА

Django сервер: http://localhost:8000

API Endpoint: http://localhost:8000/api/skills/

Статус: Все системы работают в штатном режиме

🎯 Django Backend

Полнофункциональный REST API для управления навыками с автоматическим выпуском NFT в блокчейне Ethereum

⛓️ Blockchain Integration

Каждый навык автоматически выпускается как NFT в сети Ethereum Sepolia с полной прозрачностью

🔒 Максимальная Безопасность

Приватные ключи защищены, все транзакции подписываются локально, полный контроль над данными

📡 API ENDPOINTS

GET /api/skills/
Получить все навыки текущего пользователя
POST /api/skills/
Создать новый навык (автоматически выпускает NFT в блокчейне)
GET /api/skills/{id}/
Получить конкретный навык по ID
PUT /api/skills/{id}/
Обновить навык
DELETE /api/skills/{id}/
Удалить навык

⛓️ BLOCKCHAIN INFORMATION

Сеть: Ethereum Sepolia (Testnet)

Контракт: SkillDNA.sol

Функции: Выпуск NFT навыков, отслеживание владельцев, проверка подлинности

События: SkillIssued - уведомления о выпуске навыков

Безопасность: Все транзакции подписываются локально
